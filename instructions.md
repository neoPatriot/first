# Инструкция по языку разметки Markdown

## Стилизация текста

Команда                     | Действие
------------                | ---------:
Обычный текст набираем как есть | 
Новая строка | Два перехода на новую строку
**Полужирный текст** | \*\*Текст\*\*
*Курсив* | \*Текст\*
